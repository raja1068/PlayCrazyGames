## 📱 About PlayCrazyGames

**PlayCrazyGames** is a premium collection of **9 classic arcade and board games** built with HTML5, CSS3, and vanilla JavaScript. Play instantly in your browser or install as PWA for complete offline gaming!

### ✨ Key Features:
- ✅ **100% FREE** - No hidden costs, no premium features
- 🌐 **Play Online** - No download needed for web version
- 📱 **Offline App** - Install as PWA and play WITHOUT internet
- 🎨 **Modern UI** - Beautiful gradients and smooth animations
- 📱 **Mobile Responsive** - Works perfectly on phones and tablets
- 🚀 **Fast Loading** - Optimized for quick gameplay
- 👥 **Local Multiplayer** - Play with friends on same device
- 🔫 **10-Level Shooting** - Progressive difficulty action game

## 🎮 How to Play Each Game

### 1. 🐍 Snake Game
- **Controls:** Arrow Keys or WASD
- **Objective:** Eat red food to grow and increase score
- **Game Over:** Hitting walls or your own tail
- **Feature:** Local high score tracking

### 2. ❌⭕ Tic-Tac-Toe
- **Players:** 2 (local)
- **Objective:** Get 3 marks in a row (horizontal, vertical, or diagonal)
- **Features:** Win/Draw detection, one-click reset

### 3. 🎴 Memory Match
- **Players:** 1-2
- **Objective:** Match all 8 pairs of emoji cards
- **Feature:** Move counter to track performance

### 4. ♜ Chess Game
- **Players:** 2 (local)
- **Rules:** Standard chess (pawns, rooks, knights, bishops, queen, king)
- **Features:** Check/Checkmate detection, valid move highlighting

### 5. 🐍🪜 Snakes & Ladders
- **Players:** 2-4
- **Objective:** Reach square 100 first
- **Features:** Automatic snake/ladder detection, dice rolling

### 6. 🎯 Carrom Board
- **Players:** 2
- **Objective:** Pocket all your coins into corners
- **Features:** Power control, aiming system

### 7. 🔫 Gun Shooting Game
- **Players:** 1
- **Objective:** Complete all 10 levels by shooting targets
- **Features:** Progressive difficulty, score tracking, limited ammo
- **Levels:** Each level has faster targets and more required hits

### 8. 🎲 Classic Ludo
- **Players:** 2-4
- **Objective:** Get all 4 tokens home before opponents
- **Features:** Dice rolling, token movement, capture opponents
- **Special:** Roll a 6 for extra turn and to bring out new token

### 9. ⬛ Checkers
- **Players:** 2
- **Objective:** Capture all opponent pieces or block their moves
- **Features:** King promotion, forced captures, valid move highlighting
