# 🎯 Carrom Board Game - Professional Edition

A fully functional Carrom Board Game built with Python and Pygame, featuring realistic physics, multiple game modes, themes, and save/load functionality.

## 🎮 Features

### Game Modes
- ✅ **2-Player Mode** - Classic head-to-head gameplay
- ✅ **4-Player Mode** - Play with up to 4 players
- ✅ **VS Bot Mode** - Practice against AI opponent

### Physics & Mechanics
- ✅ Realistic friction and collision physics
- ✅ Elastic rebounds off walls
- ✅ Accurate pocket detection
- ✅ Drag-and-strike controls with power meter

### Authentic Carrom Rules
- ✅ Queen pocket rule (must be covered)
- ✅ Foul penalties (striker pocketed)
- ✅ First to 29 points wins
- ✅ 4 corner pockets only

### Visual Features
- ✅ 4 Color Themes (Classic, Modern, Dark, Forest)
- ✅ 3D coin effects with shadows
- ✅ Particle effects for collisions and pockets
- ✅ Power meter with visual feedback
- ✅ Directional arrows on board

### Additional Features
- ✅ Save/Load game functionality
- ✅ Turn-based gameplay with indicators
- ✅ Score tracking for all players
- ✅ Responsive UI

## 🕹️ How to Play

1. **Drag the striker** horizontally on your baseline
2. **Release to shoot** - longer drag = more power
3. **Pocket your coins** to score points
4. **Cover the Queen** (pocket after another coin) for bonus points (50)
5. **Avoid fouls** - pocketing striker gives penalty (-5 points)
6. **First to 29 points wins!**

## 🎯 Controls

- **Mouse Drag** - Aim and power
- **ESC** - Pause game
- **S** - Save game (in pause menu)
- **L** - Load game (in pause menu)
